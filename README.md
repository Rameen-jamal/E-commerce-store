
🛒 E-Commerce Store

A full-stack e-commerce web application built for learning and production-level practice. It includes product browsing, cart functionality, authentication, and order handling with a modern UI.

🚀 Features
 Product listing & product details page
 Add to cart / remove from cart
User authentication (login/register)
 Order placement system
 Checkout flow
 Responsive UI (mobile + desktop)
Fast frontend deployment (Vercel-ready)
Backend API integration (Node/Express or similar)
 Tech Stack

Frontend:

React.js
React Router
Context API / State Management
CSS 

Backend:

Node.js
Express.js
MongoDB 
JWT Authentication

Deployment:

Frontend: Vercel
Backend: Render
